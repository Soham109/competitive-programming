# competitive-programming

My competitive programming solutions — Codeforces, CSES, and more.

### 🔗 Browse them on the website

**[cp.sohamaggarwal.com](https://cp.sohamaggarwal.com)**

The site lists every solution grouped by platform and renders the code in a
clean, IDE-style view. Each problem can have a short write-up (the idea,
approach, and complexity) shown alongside its code.

### Repo layout

```
codeforces/2218D.cpp     # the solution
codeforces/2218D.md      # optional write-up shown on the site
cses/                    # more platforms…
```

Every solution shows up on the site automatically. Adding a Markdown file
next to it (same name, or a `README.md` in its folder) gives that solution a
description — without one, the code still appears, just without notes.
