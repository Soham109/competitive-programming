# Sonya and Exhibition

> [Problem on Codeforces](https://codeforces.com/problemset/problem/1004/B)

## Idea

The key observation is that the beauty of any segment $[l_i, r_i]$ is maximized when the number of roses and lilies in that segment is as balanced as possible. This is because the beauty is the product of the two counts, and the product $x \cdot y$ is maximized when $x \approx y$ for a fixed sum $x + y$. Alternating the flower types (e.g., "010101...") ensures that every segment has the most balanced distribution of roses and lilies possible.

## Approach

1. To maximize the beauty of all segments, assign flower types in an alternating pattern (e.g., "010101..."). This ensures that any segment of length $k$ will have at most $\lceil k/2 \rceil$ flowers of one type and $\lfloor k/2 \rfloor$ of the other type, which is the most balanced distribution.

2. Construct the string by alternating between '0' (rose) and '1' (lily) for all $n$ positions. This can be implemented by toggling a boolean flag or using modular arithmetic on the index.

3. Since the problem allows multiple valid outputs, any alternating pattern starting with either '0' or '1' is acceptable.

4. Output the constructed string as the arrangement of flowers.

## Complexity

- **Time:** $O(n)$ — constructing the alternating string requires a single pass of $n$ iterations.
- **Space:** $O(n)$ — for storing the resulting string.

## Notes

- The input values $l_i$ and $r_i$ are not directly used in the solution because the alternating pattern inherently maximizes the beauty for all possible segments. This works due to the mathematical property of balanced products.
