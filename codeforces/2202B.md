# 2202B

> [Problem on Codeforces](https://codeforces.com/problemset/problem/2202/B)

## Idea

The key insight is that the string must alternate between two characters, `a` and `b`, to satisfy the problem's constraints. If the string length $n$ is even, each pair of consecutive characters must differ, except when both are `?`, which can be resolved arbitrarily. If $n$ is odd, the first character cannot be `b`, and the alternation must still hold. Additionally, the counts of `a` and `b` must not exceed their respective maximum possible counts: $\lceil n / 2 \rceil$ for `a` and $\lfloor n / 2 \rfloor$ for `b`.

## Approach

1. **Count Constraints**: First, check if the counts of `a` and `b` in the string exceed their respective maximum possible counts. If they do, output `NO` because it is impossible to construct a valid alternating string.
2. **Alternation Check**:
   - For even-length strings, verify that every pair of consecutive characters alternates (e.g., `ab` or `ba`), except for pairs of `?` which can be resolved later.
   - For odd-length strings, ensure that the first character is not `b` and that alternation is maintained for all consecutive pairs, with the same exception for `?`.
3. If all conditions are satisfied, output `YES`. Otherwise, output `NO`.

## Complexity

- **Time:** $O(n)$ per test case, as we iterate through the string once for counting and once for alternation checks.
- **Space:** $O(1)$, as we use a constant amount of extra space for counters and flags.

## Notes

- The function `even` is used to check alternation for even-length strings, while `odd` handles the additional constraint for odd-length strings.
- The check for `?` pairs ensures that they do not break alternation since they can be resolved to any valid character pair.
