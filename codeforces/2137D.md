# Replace with Occurrences

> [Problem on Codeforces](https://codeforces.com/problemset/problem/2137/D)

## Idea

The key observation is that for each distinct value $x$ in the array $a$, its frequency $f(x)$ must be divisible by $x$. This is because $x$ groups of $x$ identical elements are needed to satisfy the frequency requirement $f(x) = b_i$ for all $i$. If any $b_i$ fails this divisibility condition, constructing a valid array $a$ becomes impossible.

## Approach

1. Count the occurrences of each frequency $b_i$ using a hash map. Let `mp[x]` denote the number of times $x$ appears in the array $b$.
2. Verify the divisibility condition for all values in `mp`. Specifically, check if `mp[x] \% x == 0`. If any value fails, output $-1$ for that test case because no valid array $a$ exists.
3. For each valid value $x$ in `mp`, split `mp[x]` into $\frac{mp[x]}{x}$ groups of size $x$. Assign unique identifiers to these groups to ensure that each group corresponds to a distinct value in $a$.
4. Construct the array $a$ by repeating the group identifiers $x$ times for each group. This guarantees that $f(a_i) = b_i$ for all $i$.
5. Output the constructed array $a$ for the test case.

## Complexity

- **Time:** $O(n)$ per test case, dominated by iterating over the array $b$ and constructing $a$. Across all test cases, the total complexity is $O(\sum n) \leq 2 \cdot 10^5$.
- **Space:** $O(n)$ per test case for the hash map `mp` and the constructed array `a`.

## Notes

- The divisibility condition ensures that the grouping process is always valid for $x$ distinct elements.
