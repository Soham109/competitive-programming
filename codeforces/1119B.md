# Alyona and a Narrow Fridge

> [Problem on Codeforces](https://codeforces.com/problemset/problem/1119/B)

## Idea

The key insight is that to maximize the number of bottles $k$ that fit in the fridge, we should prioritize placing the tallest bottles first. This is because taller bottles are more restrictive in terms of the height required to store them, and placing them first ensures that the remaining space can accommodate the smaller bottles more flexibly. Additionally, bottles can only be stacked in alternating columns, so we only need to sum the heights of the tallest bottles in every other position (i.e., the odd-indexed bottles after sorting in descending order).

## Approach

1. Use binary search to determine the maximum number of bottles $k$ that can fit in the fridge. For a fixed $k$, check if the first $k$ bottles (sorted in descending order) can fit within the fridge height $h$.
2. To check if $k$ bottles can fit:
   - Extract the heights of the first $k$ bottles from the input array and sort them in descending order.
   - Since bottles can only be stacked in alternating columns, sum the heights of the tallest bottles in every other position (i.e., the odd-indexed bottles in the sorted list).
   - If this sum is less than or equal to $h$, then $k$ bottles can fit.
3. Use binary search over $k \in [1, n]$ to efficiently find the maximum $k$ for which the above condition holds.

## Complexity

- **Time:** $O(n \log n + n \log n \cdot \log n) = O(n \log^2 n)$, where $O(n \log n)$ is for sorting the array and $O(n \log n \cdot \log n)$ is for binary search over $k$ with sorting performed in each iteration.
- **Space:** $O(n)$ for storing the temporary array of the first $k$ bottles during each binary search iteration.

## Notes

- Sorting the bottles in descending order ensures that the tallest bottles are considered first, which is crucial for the greedy approach to work.
- The problem guarantees that at least one bottle can always fit in the fridge, so we can safely start the binary search from $k = 1$.
