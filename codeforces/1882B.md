# Sets and Union

> [Problem on Codeforces](https://codeforces.com/problemset/problem/1882/B)

## Idea

The key insight is that any attainable set $S$ is formed by taking the union of a subset of the input sets $S_1, S_2, \ldots, S_n$. To maximize the size of $S$ while ensuring $S \neq S_1 \cup S_2 \cup \ldots \cup S_n$, we must exclude at least one element from the universal union $U = S_1 \cup S_2 \cup \ldots \cup S_n$. The largest such $S$ is achieved by excluding one element $x \in U$ that is present in the fewest number of input sets, as this minimizes the reduction in size of $S$.

## Approach

1. Compute the universal union $U = S_1 \cup S_2 \cup \ldots \cup S_n$ and store all its elements in a set `s`. This represents the largest attainable set, which we aim to reduce by removing at least one element.

2. For each element $x \in U$, calculate the union of all input sets that do not contain $x$. This is achieved by iterating over all sets $S_i$ and including $S_i$ in the union only if $x \notin S_i$. This ensures that $x$ is excluded from the resulting set.

3. Track the size of the largest union obtained in step 2. This size represents the maximum number of elements in an attainable set $S$ such that $S \neq U$.

4. Output the maximum size found in step 3.

## Complexity

- **Time:** $O(n \cdot m \cdot |U|)$, where $n$ is the number of sets, $m$ is the average size of each set, and $|U|$ is the size of the universal union. For each element $x \in U$, we iterate through all $n$ sets and compute the union of sets that do not contain $x$, which takes $O(n \cdot m)$ time.
- **Space:** $O(n \cdot m)$ to store the input sets and intermediate unions.

## Notes

- The solution leverages the fact that $|U| \leq 50$ since the maximum element in any set is bounded by $50$. This small bound makes it feasible to iterate over all elements of $U$ and compute the unions efficiently.
- Using a `set` data structure simplifies operations like union and membership checks, but it may be replaced with a `bitmask` for further optimization in competitive settings.
